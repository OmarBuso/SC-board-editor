What is this?

This tool lets you convert between Sudoku Coach strings and their JSON representation, making it much easier to inspect, modify, and experiment with puzzle data

sudoku.coach stores puzzles as compressed text strings that usually start with SCv7_32_

These strings contain puzzle information such as:

  • Grid, regions and digits
  • Constrains
  • Cosmetic elements
  • Player progress and candidates (when present)

Why would I use it?

  • See what's actually inside a Sudoku Coach string
  • Edit puzzle definitions by hand
  • Edit constraints without the limitations of the editor
  • Debug exported puzzles
